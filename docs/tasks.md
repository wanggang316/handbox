# HandBox 开发任务清单

本文档列出了 HandBox 项目的开发任务和功能模块，按优先级和开发阶段进行组织。

## 🚀 项目概述

HandBox 是一款本地优先、隐私可控的一站式多模型 AI 工作台，基于 Tauri 2 + SvelteKit 5 + TypeScript 构建。

**技术栈:**
- 前端：Svelte 5 + SvelteKit 2 + TypeScript
- 后端：Tauri 2 + Rust
- 构建：Vite 6
- 存储：SQLite + FTS5（全文搜索）
- 安全：OS Keychain（API Key 加密存储）
